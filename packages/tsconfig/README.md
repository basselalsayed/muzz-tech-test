# @muzz/tsconfig

Shared TypeScript configurations for Muzz projects.

## Usage

Install the package and its peer dependencies:

```sh
pnpm add -D @muzz/tsconfig
```

### Node Projects

In your `tsconfig.json`:

```json
{
  "extends": "@muzz/tsconfig/node"
}
```

### React Projects

In your `tsconfig.json`:

```json
{
  "extends": "@muzz/tsconfig/react"
}
```
